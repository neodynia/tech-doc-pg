Functions

Some characteristics of the syntax of functions include:

  • No use of curly braces or semicolons.
  • A thin arrow is used to seperate the defining of the function and its steps.
  • Parameters are set equal to the function name and encased in parenthesis before the thin arrow.

Some examples of function syntax are:

          
            foo ->
              console.log('bar')
          
        
          
            aFunction = (name, age) ->
              myObj =>
                'name': name
                'age': age
                 console.log(myObj)
Objects and Arrays

Some characteristics of object-based and array-based syntax in CoffeeScript.

  • Defining each data structure does not involve var, let, const
  • Commas are usually ommited between array items and object keys and values

Some Examples of CoffeScript Syntax:

        
          items = [first second third]
        
        
        
          food =
                breakfast: muffin
                lunch: bagel
                dinner: meatloaf
Comments

Comments are:

  • Same format as Ruby. Single line comments are preceeded with a #.
  • Multiple lines of comments can be encased in ### hashes in the beginning and ### at the end of the comment.

Syntax:

          # This is a comment.

          ###
                Multiple line
                Comment
          ###
Conditionals

Conditionals:

  • Include if, else, unless, then
  • As with everything in CS, indentation is important
  • Many conditionals can be simplifed to one-liners

Some examples:

mood = greatlyImproved if eating
        
          if happy && knowIt
            clapsHands()
          else
            showIt()
        
        
        
          date = if friday then sue else jill
Loops and Comprehensions

Characteristics of Loops and Comprehensions

  • Beautiful syntax for iterations.
  • Can be filtered, which is similar to Python comprehensions.
  • Comprehensions can iterate over objects by utilizing the keyword 'of'.
  • Iterations can be written on one line using the postfix form.

Some examples: 

        
          for name in ['Ted', 'John', 'Mark']
            alert "Release #{name}"
        
      
        
          names = sam: seaborn, donna: moss
          alert("#{first} #{last}")
