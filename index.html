Functions

Some characteristics of the syntax of functions include:

  • No use of curly braces or semicolons.
  • A thin arrow is used to seperate the defining of the function and its steps.
  • Parameters are set equal to the function name and encased in parenthesis before the thin arrow.

Some examples of function syntax are:

foo -> console.log('bar')
aFunction = (name, age) -> myObj = 'name': name 'age': age console.log(myObj)
Objects and Arrays

Some characteristics of object-based and array-based syntax in CoffeeScript.

  • Defining each data structure does not involve var, let, const
  • Commas are usually ommited between array items and object keys and values

Some Examples of CoffeScript Syntax:

items = [first second third fourth fifth] food = breakfast: muffin lunch: bagel dinner: meatloaf
